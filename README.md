## Using
1. `git clone https://github.com/Mendeleev258/key-parser.git` 
1. Run **main.py**
2. Get your key
> (!) keys available for newest versions (tested on 2026.03)

## Dependencies
If you're using Linux, make sure that one of _xclip_, _xsel_, _wl_copy_ installed.
- For Ubuntu 24.04 run `sudo apt install wl-clipboard`
